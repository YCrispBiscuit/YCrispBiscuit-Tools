.env## 🚀 **项目创建方式**
### **方式1: 使用脚手架创建 (推荐新手)**
```bash
npm create vue@latest my-project
# 然后手动调整package.json中的版本号到下面指定的版本
```

### **方式2: 手动创建项目确保版本一致**
```bash
mkdir my-project && cd my-project
npm init -y  # 创建package.json文件
# 然后按下面的依赖清单安装指定版本
```

### **方式3: 直接复制现有项目 (最快最保险) ⭐推荐⭐**
```bash
# 复制整个项目文件夹
cp -r YCrispBiscuit-Tools-Client my-matrix-project
cd my-matrix-project
# 删除不需要的文件夹和功能
# 重新安装依赖
npm install
```ment用于配置VITE_开头的环境变量，如API基础URL

## 📋 **package.json scripts 建议配置**
```json
{
  "scripts": {
    "dev": "vite",
    "build": "vue-tsc -b && vite build",
    "preview": "vite preview"
  }
}
```

## ⚙️ **重要配置提醒**
- Matrix SDK需要在vite.config.ts中添加: `define: { global: 'globalThis' }`
- SCSS支持需要在vite.config.ts中配置css.preprocessorOptions
- 跨窗口通信需要确保同源策略配置正确
