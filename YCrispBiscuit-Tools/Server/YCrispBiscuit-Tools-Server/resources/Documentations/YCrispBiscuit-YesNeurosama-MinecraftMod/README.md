## 🚀 快速开始

### 📋 环境要求

- Java 21
- Minecraft 1.21（Fabric）
- 推荐使用 IDEA 或 VSCode 进行开发

### 🔧 构建与运行

1. 克隆本项目到本地
2. 使用 IDEA 或 VSCode 打开项目
3. 使用 Gradle 构建项目
4. 将生成的 mod 文件放入 Minecraft 的 mods 文件夹

## 📂 项目结构

```
YCrispBiscuit-YesNeurosama-MinecraftMod/
```
